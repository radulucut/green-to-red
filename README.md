# green-to-red
Get a color from a green to red scale

![green-To-red](./demo/green-to-red.png)

## Usage
```javascript
greenToRed(0)  // Output -> [0, 255, 0]
greenToRed(20, true)  // Output -> #32ff00
greenToRed(100, true) // Output -> #ff0000
```
